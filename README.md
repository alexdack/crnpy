# crnpy

A project to build a Python library to simulate Chemical Reaction Networks.

# To install the lib

```bash
pip install git+https://github.com/alexdack/crnpy
```

# To use the functions

```python
import crnpy
from crnpy import tools
```

# To run the tests

```bash
pip install pytest
pytest
```

# Dev

You need to run this command to update locally.

```bash
pip install .
```
